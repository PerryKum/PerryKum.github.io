• 3DS Remote Install（远程安装） 2023.08.01

教程
一、3DS Remote Install
  3DS Remote Install，即3DS远程安装，指从个人电脑利用「CIA Manager」通过局域网连接3DS的FBI来安装「.cia」文件。
  优点：不需要反复拔插存储卡（尤其是对于新三来说）；读写量比起拷贝安装少了一倍。以上两点均能有效延长存储卡寿命（3DS的存储卡实在是太容易崩了）。
  缺点：安装速度慢，只有拷贝安装的50%左右；容易因网络波动导致安装失败。
二、操作步骤
  1、确保3DS上安装了FBI。
  2、点击此处 获取CIA Manager。
  3、确保3DS与PC连接在同一局域网下，运行CIA Manager，点击「切换到FBI无线传输模式」（红圈）。
  4、3DS运行FBI，选择「Remote Install - Receive URLs over the network」，将「FBI版本」、「IP」与「Port」填入CIA Manager中的对应位置。
  5、在PC端唤出cmd（Win + R），在cmd中输入ipconfig，将「IPv4地址」填入CIA Manager中的对应位置；最后的「HTTP端口」输入1~65535的任意整数值（建议输入49152~65535），若传输因端口被占用而失败，则更改此数值即可。
  6、点击「载入一个文件夹」（黄圈），选择要上传的「.cia」文件所在的文件夹。
  7、单项传输：点击对应项后面的「开始传输」即可。
    多项传输：勾选对应项前面的复选框，然后点击菜单栏的「开始向FBI传输选中文件」即可。
  8、在3DS上确认接收。
常见问题
  一、连接超时
    原因：连接被防火墙拦截。
    解决方案：放行端口或关闭防火墙。
  二、端口被占用
    原因：HTTP端口被占用。
    解决方案：更换HTTP端口。
